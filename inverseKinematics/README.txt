In order to run this project, install the following programs

Windows:
	Visual Studio 2019
	CMake

Install CMake.

After installing CMake, set the source target to this directory (the one with the CMakeLists.txt file)
Set the bin directory to the bin folder.

Configure. 

If there are errors please refer to this tutorial:
	http://www.opengl-tutorial.org/beginners-tutorials/tutorial-1-opening-a-window/

The bin folder should now be populated and a Visual Studio solution file should exist.

Open the solution file.

If everything works, then you can begin the assignment.

For other platform installations, please refer to:
	http://www.opengl-tutorial.org/beginners-tutorials/tutorial-1-opening-a-window/
